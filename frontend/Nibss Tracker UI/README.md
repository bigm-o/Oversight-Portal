# NIBSS Ticket Tracker - Frontend

A React-based frontend application for the NIBSS Ticket Tracker system, providing analytics, governance, and audit capabilities for IT operations.

## Tech Stack
- **React 18** with TypeScript
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **Radix UI** for accessible components
- **React Router** for navigation
- **Recharts** for data visualization
- **SignalR** for real-time updates

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features
- Executive Dashboard with key metrics
- Incident Tracker with real-time updates
- Development Tracker for JIRA integration
- Analytics and reporting capabilities
- SLA compliance monitoring
- Responsive design with mobile support
